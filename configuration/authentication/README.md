# Authentication

Kafbat UI supports multiple authentication methods to secure access to the web interface and to connect to secured Kafka clusters.

## For the UI

Configure how users authenticate to access the Kafbat UI web interface:

- <a href="for-the-ui/basic-authentication.md">Basic Authentication</a> - Simple username/password authentication
- <a href="for-the-ui/oauth2.md">OAuth2</a> - OAuth 2.0 authentication with various providers (Google, GitHub, Azure, Cognito, Okta, Keycloak)
- <a href="for-the-ui/ldap-active-directory.md">LDAP / Active Directory</a> - Enterprise directory integration

## For Kafka

Configure authentication between Kafbat UI and your Kafka clusters:

- <a href="for-kafka/aws-iam.md">AWS IAM</a> - AWS IAM authentication for MSK
- <a href="for-kafka/sasl_scram.md">SASL_SCRAM</a> - SASL/SCRAM authentication
